# 基于SpringBoot+Vue的智慧政务系统
随着数字政府建设的深入推进与“互联网+政务服务”改革的纵深发展，传统政务服务模式在流程效率、服务体验与智能化水平等方面的局限日益凸显。立足于当前政务服务数字化转型的现实需求，综合运用软件工程方法论与现代全栈开发技术，设计并实现了一套架构先进、功能完备、体验优良的智慧政务服务平台。系统采用前后端分离的架构思想，后端以Spring Boot为核心，整合MyBatis-Plus、Spring Security、Spring AI及JWT等技术，构建了高内聚、低耦合的业务服务；前端基于Vue3组合式API与Pinia状态管理，结合ElementPlus组件库，构建了响应式、模块化的用户交互界面。平台创新性地将大语言模型能力通过Spring AI引入政务场景，研发了具备上下文理解能力的智能问答助手
